Profile tab

The redesign process involved crafting each page element: starting with the profile, where space was optimized for settings and a cohesive color scheme was implemented. Next, the progress bar was simplified for clarity, featuring a distinct progress number. The word of the day card was revamped to include pronunciation aids and a clickable icon. Finally, the gifts card was streamlined with a 'Copy' CTA, enhancing accessibility for users to share codes effortlessly. This iterative process aimed to create a more intuitive, user-friendly experience while maintaining a clean and minimalistic design throughout the app.

Word of the day

The process began with enhancing the app's pronunciation feature, adding alternative meanings and examples for better understanding. Next, I revamped the user interface by changing the text to 'Start Now' and integrating a 'Save' button for word recall. Then, I proposed increasing the questionnaire difficulty to extend user sessions. Finally, I designed an interactive empty box, aiming for a cleaner UI and potential drag-and-drop functionality, enhancing user experience and engagement with the app's features.

Profile Tab

While Ideating profile tab I got ideas to shift components here and there to find the perfect spot and then I started drafting wireframes. In profile tab I got five components

  • Settings
  • Progress Bar
  • Word of the day
  • Practice saved words
  • Gift referral

At first, the progress bar is not interactable and scrolling down I could see cards which is interactable and some mandate features of app. So I grabbed all the mandate cards first and decreased the size of PFP added settings near to it

Secondly, I reduced space in between cards and brought things near, so the users can actually see something new everyday. The tab is named as profile but it has every good component for a aspiring learner.

My Suggestion would be making the progress bar to minimal and giving priority to the actual features of the app

Word of the day

Word of the day is a very interesting concept but due to lack of good typeface weight it seems very much odd. I liked the whole process of learning new words by synonyms and antonyms, preferably the confetti and sounds after clicking options are very much interactive. I also had ideas to drag and drop but as the age group is above 15, It would be fancier.

The next step after tapping word of the day, the card changes to grey or disabled state and it gives some off to volume icon. After moving I got a chat image saying how to choose answers, and then those questions. I feel it very simple and it should be. As soon I answer at the first, it actually ended soon but It adds streak which is again a goal encouraging to make more use of application

My suggestion would be to add two more question to increase session length and add quality sounds and based on people how feel encouraging hearing those sounds. If possible adding vibration to wrong ones can be more interactive.

Design Suggestions

As I have discussed earlier, I have designed the pages, I have also highlighted the component with numbers and I will elaborate in detail

  1. The profile needed a revamp because, the old design wasn’t good and it took lot of space, I took the PFP and added space for settings with icon. The surface color and the primary was matching quite same. It will surely elevate a minimal and clean look and encourage users and tells user that the settings are there.
  1. The progress bar is changed because it is not an interactable component and only it can show information to the user and it give a goal oriented feel to user. I have removed the speed-o-meter and to minimal bar line. I have changed your progress number with gradient stroke so it could be distinct from the CTA
  1. I have changed the card style of word of the day and added quite a few details related to the word. As we saw a persona and as a beginner it is mandate to have the pronunciation clearly so this would surely help the user to learn better. Also there is a right icon to denote it is a clickable card
  1. I found gifts for you card slightly not matching the app but the make it more minimal and accessible I have added a CTA called ‘Copy’ for ease to copy code and refer friends

Previously we where discussing about the profile tab revamp, Here I have re designed the work for the day card,

  1. As we saw there is addon of pronunciation tags, I have further added alternatives because If a user doesn’t knew meaning but it is same of some other word and it could be easy to understand. Following this there is example in next screens, so alternatives can be best part.
  1. I have changed the text to ‘Start Now’ instead of practice which is very lame, I have added another CTA before that and It is a save button for saving words into dictionary, so that the user can recall any day
  1. To increase the session length I would suggest to increase questionnaire ranging from easy to difficult, so If the user is so beginner they can even get a few answers correct and try to achieve it daily
  1. I have added a empty box which gives a slight clean design in terms of UI and the user can tap or even drag and drop the box over there. In Figma, I couldn’t drag and drop but If it is possible then it will more fun to interact
