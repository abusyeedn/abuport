06 Suggestions

AI

This could be feasible for structured cities with zones, and even if it changes dynamically and the data changes frequently, then these mappings should also be changed. To reduce the tedious task, we can use ML algorithms to overcome these issues.

I have come up with KNN - which is mostly used to find the nearest point between two cluster of data. We can implement a K-Nearest Neighbors (KNN) classifier to find the nearest zones based on a given ward number. It works by calculating the distance between the ward co-ords and other wards in the dataset. The algorithm then identifies the K nearest data points and assigns the majority class among them to the input data point. The KNN classifier determines the nearest zones to a given ward number by considering its neighboring wards and their corresponding zones.

Code

import numpy as np
from sklearn.neighbors import KNeighborsClassifier

zones = {
    "THIRUVOTRIYUR": range(1, 14),
    "MANALI": range(15, 21),
    "MADHAVARAM": range(22, 33),
    "TONDIARPET": range(34, 48),
    "ROYAPURAM": range(49, 63),
    "THIRUVIKA NAGAR": range(64, 78),
    "AMBATTUR": range(79, 93),
    "ANNA NAGAR": range(94, 108),
    "TEYNAMPET": range(109, 126),
    "KODAMBAKKAM": range(127, 142),
    "VALASARAVAKKAM": range(143, 155),
    "ALANDUR": range(156, 167),
    "ADYAR": range(170, 182),
    "PERUNGUDI": list(range(168, 169)) + list(range(183, 191)),
    "SOZHANGANALLUR": range(192, 200)
}

ward_numbers = []
zone_names = []
for zone, wards in zones.items():
    for ward in wards:
        ward_numbers.append(ward)
        zone_names.append(zone)

X = np.array(ward_numbers).reshape(-1, 1)
y = np.array(zone_names)

knn = KNeighborsClassifier(n_neighbors=3)
knn.fit(X, y)

def classify_ward(ward_number):
    ward_number = np.array(ward_number).reshape(-1, 1)
    prediction = knn.predict(ward_number)
    neighbors = knn.kneighbors(ward_number, return_distance=False)
    nearest_zones = list(set([y[i] for i in neighbors[0]]))
    return prediction[0], nearest_zones

ward_number = int(input("Enter ward number: "))
zone, nearest_zones = classify_ward(ward_number)
print(f"Ward Number: {ward_number}")
print(f"Zone: {zone}")
print(f"Nearest Zones: {nearest_zones}")

The code sets up a K-Nearest Neighbors (KNN) classifier to predict zones based on ward numbers. It first organizes the data into arrays for training, then trains the KNN model with 3 neighbors to learn the relationship between ward numbers and zones. When a user inputs a ward number, the model predicts its zone and identifies the nearest zones using the KNN algorithm, providing a quick and accurate way to determine zones and their proximity based on ward numbers.
